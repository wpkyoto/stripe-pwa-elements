Example3: Express Checkout Element

One-click payment with Apple Pay, Google Pay, Link, and other express payment methods.

Documentation

Features of this example:

  • One-click payment with express payment methods (Apple Pay, Google Pay, Link, etc.)
  • Automatic detection of available payment methods based on browser and device
  • Simple integration with minimal code
  • Responsive design that works on all devices
  • Automatic payment confirmation with default handler

Supported Payment Methods:

  • Apple Pay - Available on Safari and iOS devices
  • Google Pay - Available on Chrome and Android devices
  • Link - Stripe's one-click checkout solution
  • PayPal - Available when enabled in Stripe Dashboard
  • Amazon Pay - Available when enabled in Stripe Dashboard

Implementation Pattern: 

// 1. Create express checkout element
const element = document.createElement('stripe-express-checkout-element');

// 2. Set required attributes
element.setAttribute('publishable-key', publishableKey);
element.setAttribute('client-secret', clientSecret);
element.setAttribute('amount', '1099');
element.setAttribute('currency', 'usd');

// 3. Append to DOM
document.getElementById('container').appendChild(element);

// 4. Wait for custom element to be defined
await customElements.whenDefined('stripe-express-checkout-element');

// 5. Listen for events (optional - default handler confirms payment)
element.addEventListener('confirm', (event) => {
  console.log('Payment confirmed', event.detail);
});

element.addEventListener('cancel', () => {
  console.log('Payment cancelled');
});

// 6. Initialize Stripe
await element.initStripe(publishableKey);

Custom Event Handling:

By default, the component automatically confirms the payment. If you need custom logic: 

// Disable default confirmation
element.setAttribute('should-use-default-confirm-action', 'false');

// Handle confirmation manually
element.addEventListener('confirm', async (event) => {
  const { detail } = event;

  // Your custom logic here
  // e.g., send to your backend, update UI, etc.

  // Complete the payment
  event.detail.complete('success');
  // or on failure:
  // event.detail.complete('fail');
});

Dynamic Updates: 

// Update amount dynamically
await element.updateElementOptions({
  amount: 2000,
  currency: 'eur'
});
