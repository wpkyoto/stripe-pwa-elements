Example4: Stripe Link Authentication Element

The Link Authentication Element collects email addresses and enables faster checkouts with Link.

Documentation

About Link Authentication Element:

  • Collects customer email addresses
  • Enables one-click checkouts with Stripe Link
  • Automatically recognizes returning Link customers
  • Can be combined with other Stripe Elements for complete checkout flows
  • Supports prefilling email addresses

Use Cases:

  • Email collection for payment flows
  • Enabling Link's one-click checkout
  • Reducing friction for returning customers
  • Building custom checkout experiences

Implementation Pattern: 

// 1. Create Link Authentication Element
const linkAuthElement = document.createElement('stripe-link-authentication-element');

// 2. Set publishable key
linkAuthElement.setAttribute('publishable-key', 'pk_test_...');

// 3. Optional: Set default email
linkAuthElement.setAttribute('default-email', 'customer@example.com');

// 4. Listen for email changes
linkAuthElement.addEventListener('linkAuthenticationChange', (event) => {
  const { detail: { email, stripe, linkAuthenticationElement } } = event;
  console.log('Customer email:', email);

  // Use the email for your payment flow
});

// 5. Get the current email value
const email = await linkAuthElement.getEmail();
console.log('Current email:', email);

Integration with Payment Element: 

// Combine with Payment Element for complete checkout
const linkAuthElement = document.createElement('stripe-link-authentication-element');
const paymentElement = document.createElement('stripe-payment-element');

// Initialize both elements
linkAuthElement.setAttribute('publishable-key', publishableKey);
paymentElement.setAttribute('publishable-key', publishableKey);
paymentElement.setAttribute('intent-client-secret', clientSecret);

// When email changes, you can update your backend
linkAuthElement.addEventListener('linkAuthenticationChange', async (event) => {
  const { detail: { email } } = event;

  // Update Payment Intent with customer email
  await fetch('/update-payment-intent', {
    method: 'POST',
    body: JSON.stringify({ email }),
    headers: { 'Content-Type': 'application/json' }
  });
});
